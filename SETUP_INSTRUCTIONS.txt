SONOJAS LANDING PAGE - EMAIL SETUP INSTRUCTIONS
==============================================

IMPORTANT: Email Configuration Required

Step 1: Generate Gmail App Password
------------------------------------
1. Go to your Gmail account: Info.sonojas@gmail.com
2. Click on your profile picture → "Manage your Google Account"
3. Go to "Security" tab
4. Enable "2-Step Verification" (if not already enabled)
5. Search for "App passwords" in the search bar
6. Click "App passwords"
7. Select "Mail" and "Windows Computer"
8. Click "Generate"
9. Copy the 16-character password (example: abcd efgh ijkl mnop)

Step 2: Update server.js file
------------------------------
1. Open server.js file
2. Find this line (around line 17):
   pass: 'your-app-password-here'
   
3. Replace 'your-app-password-here' with your generated app password
   Example: pass: 'abcdefghijklmnop'

4. Save the file

Step 3: Run the Server
----------------------
1. Open PowerShell/Terminal
2. Navigate to project folder: cd "c:\Users\pc\Desktop\landing page"
3. Run: node server.js
4. Server will start on http://localhost:3000
